Linguagem SQL

Após a modelagem do modelo conceitual e lógico é necessário o modelo fisico para a efetiva criação do banco de dados. Conheça uma linguagem para definição e manpulação de dados em sistemas de Banco de Dados. Domine os recursos da linguagem SQL, criando uma base sólida de consulta.

Realize operações relacionais em um Banco de Dados Relacional e saiba representar problemas utilizando modelo de dados.
